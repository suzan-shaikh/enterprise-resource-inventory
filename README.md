# Enterprise Resource Inventory System

## Description
An enterprise-level desktop application designed to automate corporate inventory management. Developed using **Java** (OOP) and backed by a **MySQL** database, this system provides real-time tracking of corporate assets and resources.

## Features
- **Object-Oriented Architecture:** Built using clean, scalable Java OOP concepts.
- **Database Connectivity:** Integrates seamlessly with MySQL using JDBC for secure CRUD operations.
- **Structured Workflows:** Designed based on professional Software Engineering lifecycles and structured UML models.

## Technologies Used
- **Backend:** Java (JDK 11+)
- **Database:** MySQL Server
- **GUI Components:** Java Swing / AWT
